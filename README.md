# Golang calculator using docker and simple load balancer

<p align="center">This is my first project using a containerized system like docker. For this project my professor asked me to use an ip hash based load balancer, which is not ideal for this kind of web based project.</p>

<p align="center">On this project I had to learn how to use the http protocols (even though http is not as secure as https it already is enough for this simple project), this project works only as an scratch for what kind of web based apps I can build using a containerized system and a backend manager using Go or any other language that can handle those kind of protocols.</p>
